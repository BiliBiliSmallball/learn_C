# scanf()的用法


> **scanf：**
>**① 遇空格、“回车”、“跳格”键，结束当前输入**
>**② 遇宽度结束，结束当前输入**
>**③ 遇非法输入，结束所有输入**
>**scanf的返回值表示输入成功的数的个数，若遇到输入意外返回EOF**

`scanf()` 是 C 语言中的一个标准库函数，用于从标准输入（通常是键盘）读取数据。

以下是 `scanf()` 函数的基本用法：

```c
#include <stdio.h>

int main() {
   int number;

   printf("请输入一个整数： ");
   scanf("%d", &number);

   printf("您输入的整数是: %d", number);

   return 0;
}
```

在上面的例子中，`scanf()` 函数通过 `%d` 格式化符号读取一个整数，并将其存储在变量 `number` 的地址（`&number`）中。

`scanf()` 函数的格式如下：

```c
int scanf(const char *format, ...);
```

其中：

- `format` 是一个控制字符串，包含两类字符：普通字符和格式说明符（比如 `%d`、`%s` 等）。`scanf()` 根据控制字符串来读取输入的数据，并根据控制字符串中的格式说明符将输入的数据转换成相应的类型。
- `...` 表示变长参数列表，对应的实际参数应是指针类型。这些参数接收由 `scanf()` 读取并转换的数据。

以下是一些常用的格式说明符：

- `%d`：读取整数。
- `%f`：读取浮点数。
- `%lf`：读取双精度浮点数。
- `%c`：读取字符。
- `%s`：读取字符串。
- `%u`：读取无符号整数。

注意：当使用 `scanf()` 读取字符串时，不需要使用 `&` 操作符。例如：

```c
#include <stdio.h>

int main() {
   char str[100];

   printf("请输入一个字符串： ");
   scanf("%s", str);

   printf("您输入的字符串是: %s", str);

   return 0;
}
```

请注意，`scanf()` 在处理输入时可能会遇到各种问题，比如输入类型不匹配、输入数据过长等。因此，在使用 `scanf()` 时，应该小心处理这些可能的错误情况。

#### 那么 scanf()函数的第一个参数有什么要求吗？

`scanf()`函数的第一个参数是一个格式化字符串，它指定了应该如何读取输入数据。这个字符串可以包含一些特殊的格式说明符，比如`%d`表示读取一个整数，`%f`表示读取一个浮点数，等等。这些格式说明符会被`scanf()`用来决定如何解析输入数据。

这个格式化字符串必须符合一些规则：

1. 格式化字符串必须是一个有效的 C 字符串，也就是说，它必须以空字符（'\0'）结束。
2. 格式化字符串可以包含任意数量的格式说明符，但是它们必须按照正确的顺序排列。比如，如果你先指定了一个`%d`，然后是一个`%f`，那么输入数据应该先是一个整数，然后是一个浮点数。
3. 格式化字符串中的格式说明符必须与后面的参数匹配。比如，如果你使用了`%d`，那么后面应该是一个指向整数的指针；如果你使用了`%f`，那么后面应该是一个指向浮点数的指针。
4. 格式化字符串中的普通字符（即非格式说明符）必须与输入数据中的相应字符匹配。比如，如果你的格式化字符串是`"a%db"`，那么输入数据应该是一个字符'a'，然后是一个整数，然后是一个字符'b'。

需要注意的是，`scanf()`在处理输入时可能会遇到各种问题，比如输入类型不匹配、输入数据过长等。因此，在使用`scanf()`时，应该小心处理这些可能的错误情况。
